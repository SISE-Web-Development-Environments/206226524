Q&A


Main differences between web 1.0, web2.0 and web 3.0


Web 1.0 Web 2.0 Web 3.0
Web 1.0 was constructed by very limited amount of contributors but on the other side there were many that consumed this information In Web 2.0 the content was established by users that wanted to take part in the building of the internet Web 3.0 allow to create applications by users and many companies are creating platforms in order to share between users
In Web 1.0 the search engines are fast but usually aren't accurate enough Search engine are mostly based on the Tags of the information and therefore the search engines are more accurate In Web 3.0 semantic is taken under consideration and therefore the search is based also on semantic.
Web 1.0 contains mainly static pages Web 2.0 contains dynamic content pages which are responsive Web 3.0 even more than Web 2.0 and their aim is to create personal experience of browsing

Example for search in the three versions

In Web 1.0 we will have to specify the exact URL of the page we would like to visit or we can search it in the search engine but it won't be good enough

In Web 2.0 we can search the query and will get pages that contains related tags to the query topic

In Web 3.0 we can look for the desired page by searching in the search engine which will be much more accurate since we have tags that are related to the topic.
